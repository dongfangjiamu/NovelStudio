## 使用顺序

1. 先填 `00-project-brief.md`
2. 再填 `01-genre-positioning.md`
3. 补完 `02-world-bible.md` 与 `03-character-cards.md`
4. 做出 `04-arc-outline.md`
5. 维护 `05-foreshadow-ledger.md` 与 `06-timeline.md`
6. 每写一章前，复制 `07-chapter-card.md` 到 `chapters/chXXX-card.md`
7. 根据章节卡写正文到 `chapters/chXXX-draft.md`
8. 发布前按 `09-revision-checklist.md` 做检查

## 推荐提问法

- 立项：`基于这个项目包，先帮我完成 00 和 01。`
- 设定：`读取 00 和 01，帮我补完 02 世界观 bible。`
- 人设：`基于 00 和 02，生成 03 人物卡，主角偏强成长流。`
- 卷纲：`基于 00-03，规划第一卷 20 章卷纲，写入 04。`
- 下一章：`读取 02-06，给我下一章章节卡，先不要写正文。`
- 正文：`按 chapters/ch001-card.md 写 2500 字正文。`
- 审校：`按 09 清单检查 chapters/ch001-draft.md，并给修订稿。`

## 连载更新纪律

- 每次写完新章节，都回写 `05-foreshadow-ledger.md` 和 `06-timeline.md`
- 新增人物、规则、道具时，更新 02 或 03
- 若需要改旧设定，请显式标记为 retcon
